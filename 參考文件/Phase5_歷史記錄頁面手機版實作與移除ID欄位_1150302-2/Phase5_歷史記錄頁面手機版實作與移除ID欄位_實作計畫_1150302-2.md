# 歷史記錄頁面手機版實作 & 移除 ID 欄位

## 目標

為 4 個歷史記錄頁面新增手機版 `MobileTableCard` 卡片列表、RWD 響應式 toolbar，並移除桌面版表格的 ID 欄位。

## 變更檔案

### 廠商施工歷史

#### [VendorHistoryClient.tsx](file:///Users/user/Desktop/電子白板/whiteboard-nextjs/src/app/history/vendor/VendorHistoryClient.tsx)
- 移除 ID 欄位（TableHead + TableCell）
- 新增 `Filter`、`MobileTableCard` imports
- 新增 `isFiltersOpen` 狀態
- Table 加上 `hidden md:table`
- 新增 RWD toolbar（手機版收合篩選 + 筆數 Badge）
- 新增手機版卡片列表：title=廠商名稱、subtitle=聯絡人、status=到/離院、所有欄位
