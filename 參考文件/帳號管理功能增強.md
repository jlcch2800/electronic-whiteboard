# 帳號管理功能增強

## 完成項目

### 1. 帳號鎖定狀態管理
- 在帳號編輯視窗新增「帳號狀態」區塊，顯示：
  - 失敗計次
  - 最後失敗時間
  - 鎖定至
- 新增「解除鎖定 / 清除計次」按鈕，允許管理員手動解鎖帳號

### 2. 密碼驗證規則更新
- 更新為嚴格規則：需同時包含大寫、小寫、數字及特殊符號 (共 4 種)
- 密碼最少 8 字元

### 3. 系統異動紀錄修正
- 修正密碼變更未被記錄的問題
- 修正 `user_account` 欄位顯示為 Email 的問題，現正確顯示員工編號
- 新增 `password`, `failed_attempts`, `last_failed_at`, `locked_until` 欄位的中文名稱對應

### 4. 資料庫欄位修正
- 產生 SQL 修正檔 `sql/fix_system_change_log_column_length.sql`
- 將 `system_change_log` 的 `user_account`, `user_name`, `user_unit` 欄位長度調整為 VARCHAR(30)

### 5. UI 修正
- 修正帳號編輯對話框無法捲動、儲存按鈕被遮蔽的問題
