# 表格功能統一優化

為所有資料表格頁面新增以下三項功能：
1. 每筆資料前面有選取框，支援單選/全選，按「匯出」可匯出為 xlsx
2. 選取框後面有編號欄位 (#)
3. 顯示「共 N 筆」資訊

## Proposed Changes

每個需修改的頁面都需要以下變更：

### 1. 新增狀態管理
```tsx
import { Checkbox } from '@/components/ui/checkbox'
import * as XLSX from 'xlsx'

const [selected, setSelected] = useState<Set<string>>(new Set())
```

### 2. 新增選取函數
```tsx
const toggleSelect = (id: string) => {
    const newSet = new Set(selected)
    if (newSet.has(id)) newSet.delete(id)
    else newSet.add(id)
    setSelected(newSet)
}

const toggleSelectAll = (items: any[]) => {
    if (selected.size === items.length) setSelected(new Set())
    else setSelected(new Set(items.map(i => i.id)))
}
```

### 3. 修改 Table Header - 新增 Checkbox 和 # 欄位
```tsx
<TableHeader>
    <TableRow>
        <TableHead className="w-12">
            <Checkbox
                checked={selected.size === data.length && data.length > 0}
                onCheckedChange={() => toggleSelectAll(data)}
            />
        </TableHead>
        <TableHead className="w-12">#</TableHead>
        {/* 原有欄位... */}
    </TableRow>
</TableHeader>
```

### 4. 修改 Table Body - 新增 Checkbox 和編號
```tsx
data.map((row, index) => (
    <TableRow key={row.id} className={selected.has(row.id) ? 'bg-blue-100' : ''}>
        <TableCell>
            <Checkbox
                checked={selected.has(row.id)}
                onCheckedChange={() => toggleSelect(row.id)}
            />
        </TableCell>
        <TableCell className="text-slate-400 text-sm">{index + 1}</TableCell>
        {/* 原有欄位... */}
    </TableRow>
))
```

### 5. 新增 xlsx 匯出功能
```tsx
const handleExportXlsx = async () => {
    const dataToExport = selected.size > 0 
        ? data.filter(r => selected.has(r.id)) 
        : data

    if (dataToExport.length === 0) {
        toast({ title: '無資料可匯出', variant: 'destructive' })
        return
    }

    // 轉換資料格式（依各頁面調整欄位）
    const sheetData = dataToExport.map((row, i) => ({
        '#': i + 1,
        // ... 各欄位中文對應
    }))

    const wb = XLSX.utils.book_new()
    const ws = XLSX.utils.json_to_sheet(sheetData)
    XLSX.utils.book_append_sheet(wb, ws, 'Sheet1')
    XLSX.writeFile(wb, `檔名_${format(new Date(), 'yyyyMMdd_HHmmss')}.xlsx`)
    toast({ title: '匯出成功', description: `已匯出 ${dataToExport.length} 筆` })
}
```

### 6. 新增匯出按鈕
```tsx
<Button variant="outline" onClick={handleExportXlsx}>
    <Download className="w-4 h-4 mr-1" /> 匯出
</Button>
```
