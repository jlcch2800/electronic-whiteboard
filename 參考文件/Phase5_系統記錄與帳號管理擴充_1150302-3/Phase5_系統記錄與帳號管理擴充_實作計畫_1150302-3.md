# 帳號管理 & 系統記錄功能擴充與手機版實作計畫

## 目標
因應要求，針對「帳號管理」、「系統執行記錄」、「系統異動記錄」3 個管理頁面進行以下功能擴充與調整：
1. **手機版 RWD 支援**：比照「廠商今日施工項目」，使用 `MobileTableCard` 呈現資料，隱藏非必要的系統 ID。
2. **每筆記錄編號 (`#`)**：桌面版與手機版皆需加入流水號，且分頁切換時需正確延續。
3. **明細功能**：系統執行記錄與系統異動記錄的手機版卡片需加入「明細」對話框，且內容須與桌面版一致。
4. **Excel 匯出**：
    - 系統記錄的明細對話框內支援單筆匯出。
    - 帳號管理於列表新增全域的匯出 Excel 按鈕。

## Proposed Changes

### 帳號管理 (`UserManagementClient.tsx`)
- **RWD 與手機版**：新增 `MobileTableCard` 手機視圖，利用 `isFiltersOpen` 控制過濾區塊收合。
- **編號功能**：
    - 桌面版：於表格左側新增選取與 `#` 欄位。
    - 手機版：卡片標題加上 `#序號` 格式 (如 `#1 測試管理員`)。
- **匯出功能**：新增 `handleExport` 搭配 `xlsx` 與 `file-saver`，並在列表右上角新增「匯出」按鈕。

### 系統執行記錄 (`ExecutionLogClient.tsx`)
- **RWD 與手機版**：改為 `MobileTableCard` 呈現，移除不必要的列表 ID 顯示。
- **編號功能**：
    - 桌面版：加入 `#` 流水號欄位。
    - 手機版：卡片標題加上 `#序號` 格式 (如 `#1 待處理工作項目`)。
- **明細與匯出**：於 `MobileTableCard` 的 `actionNode` 注入按鈕以開啟明細 `Dialog`。內含相同的快照及綠色「匯出」按鈕。

### 系統異動記錄 (`ChangeLogClient.tsx`)
- **RWD 與手機版**：改為 `MobileTableCard` 呈現，移除不必要的列表 ID 顯示。
- **編號功能**：
    - 桌面版：加入 `#` 流水號欄位。
    - 手機版：卡片標題加上 `#序號` 格式 (如 `#1 測試_管理員`)。
- **明細與匯出**：於 `MobileTableCard` 的 `actionNode` 注入按鈕以開啟明細 `Dialog`。內含舊/新資料比對表及綠色「匯出」按鈕。
