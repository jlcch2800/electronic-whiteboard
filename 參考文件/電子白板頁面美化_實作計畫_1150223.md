## 一、現況分析摘要

專案技術棧：**Next.js 14 + Tailwind CSS + shadcn/ui + Framer Motion + Noto Sans TC**，共 **28 個頁面路由**。

### 現有問題歸納

| 類別 | 問題描述 |
|------|---------|
| 🏗️ **架構** | 導覽列在 `HomeClient.tsx` 和 `WhiteboardClient.tsx` 中完全重複實作（~140 行），維護成本高 |

## 二、美化建議（7 大方向）

### 1. 🏗️ 導覽列重構與強化

**現況**：導覽列在首頁和 Dashboard 各自獨立實作，約 140 行重複代碼。

**建議**：
- 抽出共用 `<Navbar />` 元件，統一管理導覽邏輯
- 加入 **行動版漢堡選單**（手機/平板自動切換）
- 導覽項目加 **active 狀態指示**（底線 / 背景高亮）
- 加入 **麵包屑（Breadcrumb）** 於子頁面，讓用戶知道自己在哪
- 使用者頭像改用 **Avatar 氣泡**，hover 展開使用者資訊

### 2. 🎨 色彩系統與品牌視覺升級

**現況**：Primary 色為純黑，整體網站色彩偏冷灰，缺乏醫院品牌辨識。

**建議**：
- 建立以 **醫院藍（#1E5CB3）** 為主的品牌色彩系統
- `globals.css` 中的 CSS Variables 全面替換：
  - `--primary`：醫院藍
  - `--accent`：活力橘（搭配點綴）
  - `--success / --warning / --info`：增加語意色票
- 表格 header 加入 **品牌色漸層背景**
- 卡片、按鈕、Badge 統一使用新色彩系統
- 頁面背景改用 **極淡的漸層**，避免純白或純灰
- 增加 **Glassmorphism 效果** 於關鍵區塊（如統計卡片、導覽列）

### 3. 📊 資料表格體驗全面升級

**現況**：表格為原始 Table，無分頁、無排序、無欄位凍結，工具列擁擠。

**建議**：
- **分頁機制**：加入底部分頁器（每頁 10/20/50 筆可選）
- **欄位排序**：點擊表頭可升/降序排列
- **欄位凍結**：左側 Checkbox 和序號欄固定不動
- **行 hover 詳情**：hover 時展開完整資料預覽 Tooltip
- **表格工具列重排**：
  - 左側：搜尋區（日期 + 關鍵字）
  - 右側：操作按鈕群（新增 / 匯出 / 重整）
  - 已選取項目時，工具列顯示 **批次操作模式**（「已選 N 筆」+ 刪除 + 匯出）
- **表頭強化**：加上 sticky 效果、背景色對比、排序箭頭
- **空狀態改進**：改用插圖 + 引導文字 + CTA 按鈕

### 4. 📝 表單頁面 UX 強化

**現況**：表單為直列式卡片堆疊，缺乏引導感。

**建議**：
- **必填欄位標示**：用紅色 `*` + tooltip 說明
- **即時驗證**：欄位失焦時顯示驗證結果（✓ 綠勾 / ✗ 紅叉 + 錯誤訊息）
- **提交確認**：提交前顯示摘要確認 Dialog
- **表單 Header 強化**：加入進度指示（如「步驟 1/3」或進度條）
- 返回按鈕改為帶動畫的 **左箭頭 + 文字**，hover 有滑動效果
- 提交按鈕加入 **成功動畫**（打勾 → 跳轉）

### 5. 📱 行動裝置響應式設計

**現況**：導覽列在小螢幕溢出，表格需大幅水平捲動。

**建議**：
- 導覽列 ≤768px 切換為 **漢堡選單 + 側欄 Drawer**
- 表格 ≤768px 切換為 **卡片式列表**（每筆資料變成一張卡片）
- 表單在小螢幕自動轉為 **單欄排列**
- 統計卡片改為 **單欄堆疊 + swipeable**
- 搜尋工具列改為 **可折疊式篩選器**

### 6. 🔄 Loading 與轉場動畫統一

**現況**：Loading 只有 Loader2 spinner，各頁面不一致。

**建議**：
- 整合頁面級 **Skeleton Screen**（骨架屏），取代簡單 spinner
- 表格 loading 時顯示 **shimmer 動畫行**
- 頁面轉場加入 **fade + slide 過渡**
- 按鈕操作加入 **haptic feedback 微動畫**（如 active:scale）
- Toast 通知加入 **slide-in 動畫** + 自動消失進度條
- 刪除確認 Dialog 加入 **shake 警告動畫**
