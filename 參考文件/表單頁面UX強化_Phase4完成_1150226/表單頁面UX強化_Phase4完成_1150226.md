## 每個表單新增的 UX 功能

1. **必填欄位標記** — 紅色 `*` + hover tooltip「此欄位為必填」
2. **即時驗證回饋** — 失焦後顯示 ✓ 綠色✓ 或 ✗ 紅色✗ + 錯誤訊息（含動畫）
3. **進度指示器** — Header 內含步驟計數與動畫進度條
4. **返回按鈕** — 左箭頭 + 文字 + hover 滑動效果
5. **確認 Dialog** — 提交前顯示所有填入資料摘要
6. **成功動畫** — 提交成功後按鈕轉綠 + 打勾動畫 → 1.5 秒後自動跳轉
