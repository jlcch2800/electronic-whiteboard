# 完成報告 - 廠商施工表單與異動紀錄更新

## 1. 廠商今日施工項目表單驗證
### 修改內容
- **Schema 驗證 (`schemas.ts`)**：
  - 新增條件式驗證：當狀態為「到院」時，強制要求填寫：
    - 負責人電話
    - 棟別、樓層、施工地點
    - 工作證號、施工人數
  - 修正數字欄位 (工作證號、施工人數) 在空值時顯示 "NaN" 的問題，改用 `z.preprocess` 預處理空字串為 `null`。

- **前端頁面 (`new/page.tsx`, `edit/VendorEditClient.tsx`)**：
  - 配合 Schema 更新 UI，為必填欄位加上紅色星號 (*)。
  - 若未填寫，會在欄位下方顯示紅字錯誤訊息。
  - 修改頁面中將「到院/離院狀態」設為唯讀，防止隨意更改流程狀態。
