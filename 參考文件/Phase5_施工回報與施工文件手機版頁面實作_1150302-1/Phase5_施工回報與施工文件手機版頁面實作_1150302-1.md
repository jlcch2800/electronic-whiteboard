# 施工回報 & 施工文件 手機版頁面實作

## 變更摘要

比照 `VendorWorkClient` 的手機版卡片呈現方式，為施工回報和施工文件頁面新增 `MobileTableCard` 手機版列表和 RWD 響應式 toolbar。

### 修改檔案

| 檔案 | 變更 |
|------|------|
| [WorkReportClient.tsx](file:///Users/user/Desktop/電子白板/whiteboard-nextjs/src/app/work-report/WorkReportClient.tsx) | 新增手機版卡片列表 + RWD toolbar |
| [WorkFileClient.tsx](file:///Users/user/Desktop/電子白板/whiteboard-nextjs/src/app/work-file/WorkFileClient.tsx) | 新增手機版卡片列表 + RWD toolbar |

### 每筆卡片顯示欄位（排除 ID 和 created_at）

**施工回報**：廠商名稱（title）、負責人（subtitle）、狀態 Badge、日期、時間、地點、施工內容、備註

**施工文件**：廠商名稱（title）、上傳人員（subtitle）、「文件」Badge、日期、施工項目、說明、文件/照片/影片連結（可點擊）、備註

### 共通 RWD 優化
- 手機版篩選收合按鈕（「顯示/隱藏篩選」）
- 篩選控件全寬適配
- 操作按鈕區域自動換行
- 新增資料筆數 Badge
