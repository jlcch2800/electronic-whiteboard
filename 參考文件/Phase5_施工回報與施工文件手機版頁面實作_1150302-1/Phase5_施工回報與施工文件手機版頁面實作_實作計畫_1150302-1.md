# 施工回報 & 施工文件 手機版頁面實作

比照 `VendorWorkClient` 的手機版卡片呈現方式，為 `WorkReportClient` 和 `WorkFileClient` 加入 `MobileTableCard` 手機版列表，並優化 toolbar 的 RWD 響應式佈局。

## 變更內容

### WorkReportClient

#### [WorkReportClient.tsx](file:///Users/user/Desktop/電子白板/whiteboard-nextjs/src/app/work-report/WorkReportClient.tsx)

1. **Import `MobileTableCard`** 和 `Filter` icon
2. **加入 `isFiltersOpen` state** 控制手機版篩選展開/收合
3. **桌面版 Table 加上 `hidden md:table`**，使其在手機版隱藏
4. **Toolbar 區域改為 RWD**：手機版有收合篩選按鈕，展開後顯示日期、狀態、關鍵字等篩選條件
5. **加入手機版卡片列表** `<div className="md:hidden">`，每筆卡片顯示以下欄位（排除 ID 和 created_at）：
   - title: `vendor_name`（廠商）
   - subtitle: `engineering_contact`（負責人）
   - status: `work_status` 的 Badge（完成/未完成/異常）
   - date: `report_date`
   - time: `report_time`
   - details: 地點、施工內容、備註

## 驗證計畫

### 瀏覽器手動驗證
1. 開啟 `http://localhost:3000/work-report`，縮小瀏覽器至手機寬度（< 768px），確認卡片列表正確顯示且所有欄位皆有呈現
2. 開啟 `http://localhost:3000/work-file`，同樣操作確認
3. 確認桌面版（> 768px）時仍顯示原本的表格、手機版卡片列表隱藏
4. 測試篩選功能在手機版的展開/收合行為
