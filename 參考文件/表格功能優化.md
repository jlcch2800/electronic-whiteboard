已為 8 個表格頁面加入統一功能：

| 頁面 | Checkbox | # 編號 | xlsx 匯出 | 筆數顯示 |
|------|:--------:|:------:|:---------:|:--------:|

| EngineeringHistoryClient | ✅ | ✅ | ✅ | ✅ |
| PendingHistoryClient | ✅ | ✅ | ✅ | ✅ |
| ReportHistoryClient | ✅ | ✅ | ✅ | ✅ |
| WorkReportClient | ✅ | ✅ | ✅ | ✅ |
| WorkFileClient | ✅ | ✅ | ✅ | ✅ |
| ChangeLogClient | ✅ | ✅ | ✅ | ✅ |
| ExecutionLogClient | ✅ | ✅ | ✅ | ✅ |

> [!NOTE]
> `UserManagementClient` 為單選編輯設計，不適合多選匯出功能。

## 功能說明

1. **Checkbox 多選**：表頭全選 + 各列單選
2. **# 編號欄**：根據頁面順序自動編號
3. **xlsx 匯出**：
   - 勾選資料 → 匯出勾選項
   - 無勾選 → 匯出全部篩選結果
   - 檔名格式：`{類別}_{yyyyMMdd_HHmmss}.xlsx`
4. **筆數顯示**：`共 X 筆` Badge

## 修改/刪除按鈕樣式統一

將以下頁面的操作按鈕改為與「廠商今日施工」一致的樣式：

| 頁面 | 說明 |
|------|------|
| WorkReportClient | 施工回報管理 |
| WorkFileClient | 施工文件管理 |

### 新樣式特點
- 移除每列的「操作」欄位圖示按鈕
- 在搜尋列加入「修改」和「刪除」按鈕
- **修改**：選取 1 筆時可用（跳轉編輯頁）
- **刪除**：選取 ≥1 筆時可用（紅色樣式 + 確認對話框）
